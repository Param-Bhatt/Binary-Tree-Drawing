Implementation of Tidy and Tidier drawing Algorithms
  • Line drawing

    • Midpoint line drawing algorithm goes as follows: two functions are made handling two cases each. One of them handles the cases if the absolute value of the slope is less than one and the x-coordinate of the first point is less than that of the second point. The other is similar to the first except it handles the cases if the absolute value of the slope is greater than one. Then another function which handles all the cases, exchanges the points if needed and calls the correct function to draw the line.


    This algorithm is implemented using the class "line".
    The class is declared in "line.h" and all the member functions have been defined in the file "line.cpp".

    Line class

      Private elements
    1. int sx

      This is the x-coordinate of the starting point of the line.
    2. int sy

      This is the y-coordinate of the starting point of the line.
    3. int ex

      This is the x-coordinate of the ending point of the line.
    4. int ey

      This is the y-coordinate of the ending point of the line.
      Public Elements

    1. Constructor

      This is the constructor function of the class "line".It initializes the objects of the class whenever there is an instance of the class in our program.

    2. void draw()

      It calculates the values of the co-ordinates onthe line and renders them with double the width and a black colour.
  • Circle drawing

    • Midpoint circle drawing uses the 8-way symmetry from the horizontal and vertical axes as well as the y = x and y = -x axes about the center of the circle. The basic circle drawing algorithmis a bit modified to handle the position (or center) of the circle. Thus the circle drawing calculates all the pixels with respect to the origin but when plotting the pixels, all the pixels are plotted by a shift of the x and y coordinates of the center of the circle which is passed during the call.


    This algorithm is implemented using the class "circle".
    The class is declared in "circle.h" and all the member functions have been defined in the file "circle.cpp".
